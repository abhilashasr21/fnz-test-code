# Deployed Code Information


## Subscription Vending

Folder Conatines a code for the Subscription Vending.

### Resources

- `Budget` : Can be used to create the budget. Once the budget is reached there can be notification send emails to the assigned users.
- `Management Group` : It will be used to create the Management Group for Oragnization.
- `Role Assignment` : Role Will be assigned to the SUbscription as per the Privilages assigned.
- `Subscription` : This Module can be used to create subscription with the proper billing information.
- `Subscription Association` : Once the Subscription is created it needs to be placed at the proper placement in the Managemnt Group Hirearchy.
